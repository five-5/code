# 树
树结构本身是一种天然的组织结构
</br>eg. 图书馆的组织->书、公司的组织->员工
</br> 高效，可以很快地查找

# 树的分类及其特点
## 一、二叉树
具有天然的递归结构&nbsp;左子树也是二叉树&nbsp;右子树也是

## 二、二分搜索树
1. 二叉树
2. 每个结点的值：大于其左子树的所有结点值，小于其右子树的所有结点值
3. 每个子树也是二分搜索树
4. 存储的元素必须有可比性


### 实现
1. 添加：非递归写法，和链表很像
2. 查询
3. 遍历 [在线性结构下，遍历是容易的]
</br> 中序遍历得到排序之后的结果
</br> 后续遍历的应用：释放内存
4. 删除 [拿到结点的前驱和后继]
