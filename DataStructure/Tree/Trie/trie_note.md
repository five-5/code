# Trie[前缀、字典树]
一般用来处理字符串

## 概要讲解
**字典**

如果有n个条目，使用树结构，查询的时间复杂度为O(logn)，100万个条目（2^20）

**Trie**

查询每个条目的时间复杂度只与查询单词的长度相关O(w)，大多数单词长度不超过10个

## 结点定义
每个节点有26(n)个指向下个节点的指针（考虑不同的语言，不同的情境）
``` c++
class Node{
    // 是否是单词的结尾
    bool isWord;
    // 只能处理26个字母
    // Node next[26];
    // 处理内容更灵活
    Map<char, Node> next;
};
```
